# Contributing to color_fields

## Quick Checklist

- [ ] JSON files are valid
- [ ] New palettes have hex colors and 4+ style_tags
- [ ] New color spaces have both GLSL and WGSL conversion functions
- [ ] Entries added to appropriate `index.json`

## High Value Contributions

- New semantic palettes with distinct moods
- Additional cosine palette presets
- New color spaces (CIE Lab, Jzazbz, ICtCp)
- Post-processing effects (chromatic aberration, dithering, film grain)
- Perceptual palette generation algorithms
